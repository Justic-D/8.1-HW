## Задание 3 SQL vs NoSQL

3.1 Напишите пять преимуществ SQL-систем по отношению к NoSQL.

3.1* Какие, на ваш взгляд, преимущества у NewSQL систем перед SQL и NoSQL.

***Приведите ответ в свободной форме.***

## Ответ:

3.1 Приемущиства:  
   - Вертикальная масштабируемость;
   - Реплицируемость;
   - Нормализация;
   - Томичность;
   - Общедоступность;

3.1* Приемущества:

Ну это некое обьединение подходов - ввод новой реализации в традиционные реляционные базы данных. Т.е. объединение преимущества SQL 
и NoSQL;
   - Легко переключаться между типом и потребностями пользователя; 
   - Оптимизация обработки онлайн-транзакций;
   - Масштабируемость;
   - Гибкость;
   - Бессерверная архитектура. 
  
При этом структуры NewSQL, как и реляционные базы данных, поддерживают принципы ACID и согласованность. 
Они могут масштабироваться, зачастую по запросу, не влияя на логику приложения и не нарушая модель транзакций.
